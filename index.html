3D Object Recognition in Real-Time

September 2023 - November 2023
Python YOLO Computer Vision TensorFlow

Project Overview

Developed a sophisticated real-time detection system capable of identifying and tracking humans and vehicles in complex environments. The system processes video inputs and provides accurate detection with bounding boxes and confidence scores.

Key Features

  • Real-time object detection with high accuracy (over 90% on test datasets)
  • Multi-object tracking capabilities with unique ID assignment
  • Performance optimization for processing up to 30 frames per second
  • Customizable detection parameters for different environments
  • Data logging and analysis of detection results

Technical Implementation

Utilized YOLO (You Only Look Once) algorithm implemented in Python with TensorFlow backend. The system was designed with a modular architecture allowing for easy integration with other systems. OpenCV was used for image processing and visualization of detection results.

Challenges & Solutions

Faced challenges with detection accuracy in poor lighting conditions, which was resolved by implementing adaptive contrast enhancement. Performance optimization required significant effort to maintain real-time processing capabilities while preserving detection accuracy.

Text-to-Speech System

January 2024 - April 2024
Python PyTorch Deep Learning NLP

Project Overview

Created an advanced deep learning-based text-to-speech conversion model capable of generating natural-sounding speech from written text. The system supports multiple voice types and can adapt to different speaking styles.

Key Features

  • High-quality speech synthesis with natural intonation and rhythm
  • Support for multiple languages including English and Vietnamese
  • Voice customization with different gender and age characteristics
  • Emotion detection and expression in synthesized speech
  • Real-time processing for immediate speech output

Technical Implementation

Implemented using PyTorch with a sequence-to-sequence architecture featuring attention mechanisms. The model was trained on over 100 hours of speech data to ensure high-quality output. Wavenet-inspired techniques were used for the vocoder component to generate realistic waveforms.

Challenges & Solutions

Achieving natural-sounding speech required extensive experimentation with different model architectures. Particular attention was paid to prosody modeling to capture the rhythm and intonation of natural speech. Performance optimization was necessary to enable real-time processing on consumer hardware.

Product Management System

October 2024 - December 2024
C# .NET Blazor Data Analytics

Project Overview

Engineered a comprehensive Blazor-based inventory system with robust data analytics capabilities for TKELOG. The system enables businesses to track and manage inventory in real-time while providing valuable insights through data visualization and reporting tools.

Key Features

  • Real-time inventory tracking and management
  • Automated alerts for low stock levels and reorder points
  • Comprehensive reporting dashboard with customizable metrics
  • Data visualization tools for trend analysis and forecasting
  • User role management with customizable permissions
  • Integration capabilities with other business systems

Technical Implementation

Developed using C# and .NET with Blazor framework for the frontend. The application follows Object-Oriented Programming principles with a layered architecture ensuring separation of concerns. Entity Framework Core was used for database operations, and SignalR for real-time updates.

Challenges & Solutions

Creating an intuitive user interface while maintaining robust functionality presented significant challenges. These were addressed through iterative design processes and user testing. Performance optimization was required for handling large datasets, which was achieved through efficient database design and query optimization.
